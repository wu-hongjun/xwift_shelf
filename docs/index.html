Xwift Shelf

Automatic Maya Python toolkit development pipeline.

        
        playground.py >>>

A flexible and powerful shelf with an automatic script deployment and reload mechanism makes developing for Maya so much easier.

    Traditional Workflow

  • Write raw code in Script Manager with no aid.
  • Save all scripts in the very hard to find Maya /scripts path.
  • Manually reload every time you make a change in code.
  • Email your script to artist and hope they know how to setup.
  • Make every button on the custom shelf by hand.
  • One shelf button can only call one script.
  • All the scripts you ever wrote are scattered everywhere.

    Life With Xwift

  • Develop in PyCharm with all the IDE benefits.
  • Manage and store all of your tools using Git repository.
  • Automatic codebase reload for every execution.
  • One click to deploy all the scripts into Maya.
  • Xwift shelf builds automatically after starting Maya.
  • You can fit as many tools into a button as you wish.
  • All tools are organized in one shelf with clean hierarchy.

Powerful Shelf

Xwift Shelf is a flexible Maya shelf written in Python 3 with sub-list and icon support.

Automatic Reload

Xwift will automatically deploy and refresh the codebase in background each time you run code.

Complex Tool Support

Xwift can handle large and complex toolkit deployment with ease.

Fully Customizable

Xwift Shelf is fully customizable and open source, so you can hack it to how you want it to be.

Easy Deployment

Xwift Shelf is super easy to deploy and setup, making tool transfer a breeze.
